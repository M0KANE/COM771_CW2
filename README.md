# COM771 Threat Analysis Project

This project contains a Jupyter Notebook titled `COM771_threatanalysis.ipynb`, which focuses on data analysis, machine learning, and visualization related to cyber security threats. The notebook includes various libraries for data manipulation, model training, and exploratory data analysis.

## Overview

The `COM771_threatanalysis.ipynb` notebook is designed to help users understand and analyze cyber security threats through data-driven approaches. It covers the following key areas:

- **Data Loading and Inspection**: The notebook begins by loading a dataset related to cyber security threats and inspecting its structure.
- **Data Cleaning and Preprocessing**: It includes steps for cleaning the data, handling missing values, and preparing the dataset for analysis.
- **Exploratory Data Analysis (EDA)**: Various visualizations are created to explore the distribution of attack types and protocols, helping to identify patterns and insights.
- **Machine Learning**: The notebook implements several machine learning models to classify attack types based on the features in the dataset.

## Usage Instructions

1. **Prerequisites**: Ensure you have Python installed along with the necessary libraries. You can install the required libraries using pip:

   ```
   pip install pandas numpy matplotlib seaborn plotly scikit-learn wordcloud
   ```

2. **Running the Notebook**: Open the `COM771_threatanalysis.ipynb` file in Jupyter Notebook or any compatible environment. You can run the cells sequentially to execute the analysis.

3. **Understanding the Results**: The notebook provides visualizations and model performance metrics that help in understanding the effectiveness of different machine learning models in classifying cyber security threats.
